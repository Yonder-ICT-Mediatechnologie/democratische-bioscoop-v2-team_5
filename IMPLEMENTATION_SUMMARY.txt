HOW TO USE
================================================================================

1. OPENING THE APPLICATION:
   - Open /sessions/funny-keen-ramanujan/mnt/Popvote/index.html in browser
   - No build process or dependencies needed
   - All files are self-contained

2. BROWSING FILMS:
   - Homepage loads with 6 films
   - Filter by genre or scheduled date
   - Click film card for details
   - Click "Details" button for full page
   - Vote button available on all pages

3. VOTING:
   - Go to "Stemmen" page
   - See ranked list of films
   - Click vote button to add your vote
   - Ranking updates in real-time
   - Only Friday films shown

4. RESERVING SEATS:
   - Click reserve from film detail or use direct link
   - Select seats by clicking (green highlight)
   - Enter name and email
   - Review summary
   - Submit to get confirmation
   - Confirmation shows ID and all details

5. ADMIN ACCESS:
   - Go to Dashboard page
   - Login: admin / admin123
   - Manage films (add, edit, delete)
   - View all reservations
   - Logout button available

6. CONNECTING REAL API:
   - Update baseURL in js/api.js
   - Set api.useMockData = false;
   - API will be called instead of mock data
   - All endpoints ready to use
